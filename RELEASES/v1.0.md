# Azazel Launcher

A custom Android home-screen launcher with a **holographic OpenGL ES 3.0 sphere**, a cyberpunk aesthetic, and an animated lock-screen overlay.

---

## Features

| Feature | Description |
|---|---|
| **Holographic Sphere** | Full-screen OpenGL ES 3.0 sphere with fiery plasma inner layer, cyan grid outer layer, emission rings, and particle sparks — all animated in real time. |
| **App Drawer** | Horizontal RecyclerView at the bottom of the screen listing all installed apps. Icons are loaded asynchronously and cached to prevent jank. |
| **Holographic Icon Frames** | Each app icon is layered on top of a cyan radial-gradient oval background for the holographic look. |
| **Lock Screen Overlay** | A `WindowManager` overlay service that draws the sphere, a pulsing fingerprint scanner, and scrolling hex IDs on top of the lock screen. |
| **Video Background (optional)** | Drop `jinx.mp4` into `res/raw/` and uncomment one block in `LockScreenOverlayService` to replace the sphere with a looping video background.
